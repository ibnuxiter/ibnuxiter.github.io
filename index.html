Boost Performa Game-mu dengan Cortiez

Cortiez adalah project gabungan dari IbnuXiter dan CyrustMods. Dibangun dengan Shizuku untuk pengelolaan performa, overlay, dan konfigurasi game yang aman.

Cortiez juga menyediakan fitur manajemen file ke direktori: Android/data/com.dts.freefireth untuk pemasangan konfigurasi atau aset personal.

info Tentang Project

Kolaborasi dua developer: IbnuXiter & CyrustMods. Fokus pada stabilitas dan kontrol perangkat, bukan modifikasi game.

Fitur Utama

rocket Game Booster

Optimasi RAM, CPU, dan jaringan tanpa memodifikasi game.

speed FPS Monitor

Overlay FPS aman, tidak menyentuh file game.

tune Sensitivitas Presisi

Pengaturan sensitivitas cerdas & responsif.

my_location Crosshair Overlay

Crosshair tampilan layar (overlay), aman dan tidak injeksi ke game.

folder_open Manajemen File

Dapat memasukkan file konfigurasi ke Android/data/com.dts.freefireth.
